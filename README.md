Comment System (reporting & moderation):
Implemented a reporting system allowing users to flag inappropriate comments. Admins can filter by reported comments to quickly identify violations, and users retain the ability to delete their own comments.

Profile History (persistent records):
Profile deletion is not permanent. A deleted profile history ensures admins can still access information from removed or inactive accounts, whether deleted by the user or an admin.

Real-Time Search:
Search bars update results dynamically with each keystroke—no need to press enter—providing instant filtering across relevant fragments.

Image Caching (performance optimization):
Images are cached on first load, reducing repeated fetches and significantly improving load times and overall UI responsiveness.
